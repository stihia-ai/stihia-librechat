# Support

## How to get help

- **Bug reports** — Open a [Bug Report](https://github.com/stihia-ai/stihia-librechat/issues/new?template=bug_report.md)
  using the issue template.
- **Feature ideas** — Open a [Feature Request](https://github.com/stihia-ai/stihia-librechat/issues/new?template=feature_request.md).
- **Security issues** — Follow the process in [SECURITY.md](SECURITY.md).
  Do not open a public issue.
- **General questions** — Open a [Discussion](https://github.com/stihia-ai/stihia-librechat/discussions)
  (once enabled) or a regular issue.

## Response times

- **Issues:** Triaged within 7 days.
- **Pull requests:** Initial review within 14 days.
- **Security reports:** Acknowledged within 3 business days.

## Scope

This repository covers the Stihia guardrail proxy and its Docker Compose
deployment. For issues with LibreChat itself, please file at
[danny-avila/LibreChat](https://github.com/danny-avila/LibreChat). For the
Stihia guardrail engine (Stihia API and `stihia` Python SDK package), contact **<support@stihia.ai>**.
